About Me

Frontend Developer

I'm David Fasakin — a passionate frontend developer who brings digital ideas to life through clean, responsive, and user-focused web interfaces. I specialize in HTML, CSS, and JavaScript, and I'm actively building with frameworks like React and exploring mobile development with React Native.

My journey into web development started with a desire to blend creativity and logic — now, I use code to solve real-world problems, improve user experiences, and build interfaces that don't just look good but work flawlessly across all devices.

I'm constantly learning, iterating, and pushing boundaries. Whether it’s crafting intuitive layouts or optimizing performance, I take pride in writing clean, maintainable code. I’m currently seeking internship opportunities where I can contribute, grow, and collaborate with developers solving meaningful challenges.

Services

Responsive Design

I create websites that look great and work seamlessly on all screen sizes from desktops to mobile phones using HTML5 and CSS3 media queries.

Frontend Development

I build clean, interactive user interfaces with HTML, CSS, and JavaScript. I write organized code and focus on speed, usability, and cross-browser compatibility.

Project Deployment

I deploy my projects using GitHub Pages, making them accessible online. I understand basic deployment practices and file structuring.
